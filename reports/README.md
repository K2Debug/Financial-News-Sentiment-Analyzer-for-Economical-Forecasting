# Reports

Academic and tool reports for EF-02, plus generators and figure assets.

## Generators

```bash
# From this folder (needs research/data/processed populated)
py generate_report.py

# Tool / dashboard report (uses images/)
py generate_tool_report.py
```

| Path | Contents |
|------|----------|
| `figures/` | Chart PNGs used in the academic report |
| `images/` | Screenshots for the tool report |
| `*.docx` / `*.pdf` | Generated or edited report drafts |

Paths assume this layout: research data lives in `../research/data/processed/`.
